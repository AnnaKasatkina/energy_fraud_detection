# Energy Meter Fraud Detection

Проект посвящён задаче выявления мошенничества в показаниях счётчиков электроэнергии по временным рядам индивидуального потребления и зашумлённому агрегированному измерению.

Предполагается, что некоторые счётчики занижают свои показания с неизвестным коэффициентом, в то время как общий счётчик фиксирует истинную суммарную нагрузку с шумом. Для обнаружения мошенников используются методы наименьших квадратов и их модификации.

## Реализованные подходы

- **Известное число мошенников**  
  Ограниченный МНК с перебором подмножеств.

- **Неизвестное число мошенников**  
  Полный МНК с пороговой классификацией коэффициентов.

- **Гибридный метод**  
  Комбинация предварительной фильтрации и локального перебора для улучшения масштабируемости.

## Эксперименты

Проведены численные эксперименты для групп из 5–20 счётчиков с различным числом мошенников и шумом до 10%. Гибридный метод показал наиболее устойчивые результаты при росте размерности задачи.

## Технологии

- Python
- NumPy, Pandas
- Численные и статистические методы
